Sportlytics

API Documentation

Sportlytics provides a public API to fetch leaderboard standings and public athlete profiles. Base URL: http://localhost:5000/api

1. Get Public Leaderboard

GET /public/leaderboard?sport=football

[
  {
    "_id": "user_id_here",
    "name": "Alex Rivera",
    "efficiencyScore": 125.4,
    "totalMatchesLogged": 12,
    "sports": ["football", "running"]
  }
]

2. Get Public Profile

GET /public/profile/:userId

{
  "_id": "user_id_here",
  "name": "Alex Rivera",
  "bio": "Amateur forward.",
  "efficiencyScore": 125.4,
  "badges": [{ "name": "First Blood", "icon": "local_fire_department" }]
}
